# Núcleo Saúde - Sistema de Gerenciamento de Planos de Saúde

## Descrição do Projeto

O **Núcleo Saúde** é um sistema de gerenciamento de clientes e planos de saúde desenvolvido na linguagem **C**. O projeto tem como objetivo facilitar o cadastro, consulta, atualização e remoção de clientes, além de realizar o cálculo automático dos valores dos planos de acordo com regras de negócio previamente definidas.

O sistema foi criado com foco no aprendizado de conceitos fundamentais da programação estruturada, utilizando manipulação de arquivos para armazenamento permanente dos dados e proporcionando uma experiência simples e eficiente através de menus interativos no terminal.

## Tecnologias Utilizadas

- **Linguagem C**: Desenvolvimento da aplicação.
- **Structs**: Organização dos dados dos clientes e dependentes.
- **Manipulação de Arquivos Binários**: Armazenamento persistente dos registros.
- **Funções**: Modularização e reutilização do código.
- **Lógica de Programação**: Implementação das regras de negócio.

## Funcionalidades

- Cadastro de clientes
- Cadastro de dependentes
- Atualização de dados cadastrais
- Remoção de clientes
- Listagem de clientes cadastrados
- Cálculo automático do valor do plano
- Validação de dados de entrada
- Armazenamento permanente em arquivo

## Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/paulocesargit/nucleosaude.git
```

2. Acesse o diretório do projeto:

```bash
cd nucleosaude
```

3. Compile o programa:

```bash
gcc main.c -o nucleosaude
```

4. Execute o sistema:

```bash
./nucleosaude
```
